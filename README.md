# Do It Now – Motivation Tracker

An Android productivity app that helps users set goals, track progress, and stay motivated.

## Features (Planned)
- Goal creation with deadlines
- Daily progress tracking
- Streak system
- Motivational reminders
- Ranking system

## Tech Stack
- Kotlin
- Jetpack Compose
- Firebase (planned)
- MVVM Architecture
